# Outdoor Gear Note

Amazon Outdoor Recreation / Camping & Hiking content pipeline for Hugo.

This project mirrors the structure of `pet_gear_note`, but is tuned for outdoor gear, camping equipment, tents, hiking accessories, and related Amazon BSR leaf categories.

## What This Project Does

- Reads `data/outdoor_camping_bsr_urls.json`
- Finds true leaf categories from the crawler output
- Stores task status in `data/tracking.json`
- Uses AutoCLI to fetch Amazon bestseller pages and product detail JSON
- Uses an OpenAI-compatible LLM to generate Hugo Markdown articles
- Generates stable SEO titles, meta descriptions, keywords, FAQ sections, clean slugs, and contextual internal/resource links
- Writes generated posts under `content/camping-hiking/` or `content/outdoor-recreation/`

No article content is generated by default. Run `--sync-only` first to initialize tasks.

## Setup

```bash
cd D:\project\outdoor_gear_note
python -m venv .venv
.venv\Scripts\activate
pip install -r requirements.txt
copy .env.example .env
```

Edit `.env`:

```env
LLM_API_KEY=sk-your-key-here
LLM_BASE_URL=https://dashscope.aliyuncs.com/compatible-mode/v1
LLM_MODEL=qwen-plus
```

## Initialize Category Tasks

```bash
python main.py --sync-only
```

This creates `data/tracking.json` from the crawler output without scraping products or generating articles.

## Run A Small Batch

```bash
python main.py --batch-size 4 --reset-processing
```

Useful options:

```bash
python main.py --batch-size 2 --top-n 20 --min-products 10
python main.py --retry-failed --batch-size 4
python main.py --refresh-seo-only
python main.py --refresh-links-only
```

## Notes

- Camping and tent-related leaf categories are prioritized first.
- Purchase links are sanitized to clean `https://www.amazon.com/dp/{ASIN}` links.
- Exact prices, coupons, sale language, and affiliate parameters are stripped from generated Markdown.
- The prompt is written for professional outdoor buying guides and Google-first helpful content, but it avoids fake hands-on testing claims.
- FAQPage structured data and a filtered sitemap are generated through Hugo layouts.
- Set `baseURL` in `hugo.toml` to the real production domain before submitting the sitemap in Google Search Console.
